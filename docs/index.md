# DonK-Kong

## 1.あらすじ
>> 時は20XX年、サイバーゴリッラて世界中の女性がさらわれてしまった。  
>> そこで立ち上がったサイバー配管工M〇RI〇。
>> 今、一人のサイバー配管工の戦いが始まる。
